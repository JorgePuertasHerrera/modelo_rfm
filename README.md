# 📊 Análisis de Segmentación de Clientes (Modelo RFM)
🧠 Descripción

Este proyecto consiste en el desarrollo de un dashboard interactivo en Power BI basado en el modelo RFM (Recency, Frequency, Monetary) para segmentar clientes según su comportamiento de compra.

El objetivo es analizar cómo interactúan los clientes con el negocio, identificar los segmentos más valiosos y extraer insights que apoyen la toma de decisiones.

🎯 Objetivos
Segmentar clientes según su comportamiento de compra
Identificar los grupos de mayor valor
Analizar métricas clave como ingresos, frecuencia y valor de vida del cliente (LTV)
Entender el impacto del comportamiento del cliente en el rendimiento del negocio
📂 Dataset
+50.000 registros de transacciones
Más de 4.300 clientes únicos
Incluye fechas de compra, valor de transacciones e identificadores de cliente
🛠️ Herramientas y tecnologías
Power BI (modelado de datos y visualización)
Excel (preparación inicial de datos)
DAX (creación de medidas y métricas)
📊 Funcionalidades del dashboard
Segmentación de clientes mediante modelo RFM
Visualización de KPIs (Ingresos, LTV, Ticket medio)
Análisis temporal de ingresos y clientes
Distribución de clientes por segmento
Interpretación de datos orientada a negocio
🔍 Principales insights
El valor del cliente está más relacionado con la frecuencia de compra que con el gasto puntual
Los clientes VIP generan mayor valor acumulado a largo plazo
El crecimiento del negocio depende en gran medida de la retención y recurrencia
🚀 Recomendaciones de negocio
Implementar estrategias de fidelización para clientes VIP
Incentivar la recurrencia en clientes frecuentes
Diseñar campañas de reactivación para clientes ocasionales
Identificar productos que generen hábitos de compra
⚠️ Retos y aprendizajes

Durante el desarrollo se detectaron inconsistencias en el cálculo de ingresos debido a problemas de agregación entre Excel y Power BI.

Esto implicó:

Revisar la granularidad de los datos (cliente vs transacción)
Ajustar las medidas para evitar duplicidades
Validar resultados con los datos originales

👉 Este proceso reforzó la importancia del modelado de datos y la validación de métricas.
